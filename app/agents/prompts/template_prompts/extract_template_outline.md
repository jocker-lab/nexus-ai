# 模版大纲提取

你是一个专业的文档分析专家。请仔细分析以下文档内容，提取其作为写作模版的结构大纲。

## 文档内容

```
{{ document_content }}
```

## 提取要求

请从上述文档中提取以下信息：

### 1. 基础信息
- **title**: 模版标题（简洁明了，能反映文档类型）
- **summary**: 模版用途摘要（50-100字，说明这个模版适合用来写什么类型的文档）
- **category**: 分类（如：市场分析、信用评级、行业研究、可行性报告、尽职调查、年度报告等）

### 2. 写作参数
- **writing_style**: 写作风格
  - `academic`: 学术风格（适用于研究论文、学术报告）
  - `business`: 商务风格（适用于商业报告、企业文档）
  - `technical`: 技术风格（适用于技术文档、规格说明）
  - `creative`: 创意风格（适用于营销文案、创意内容）
  - `journalistic`: 新闻风格（适用于新闻稿、行业资讯）

- **writing_tone**: 写作语气
  - `formal`: 正式（官方文件、法律文书）
  - `neutral`: 中性（客观报告、分析文档）
  - `casual`: 随意（博客、内部沟通）
  - `professional`: 专业（行业报告、专家意见）
  - `persuasive`: 说服性（提案、销售文档）

- **target_audience**: 目标受众（如：投资者、管理层、技术人员、普通读者）

### 3. 章节结构
分析文档的结构，提取主要章节信息。每个章节应包含：
- **title**: 章节标题
- **description**: 章节描述/写作指导（该章节应该包含什么内容）
- **estimated_percentage**: 预估占比（该章节在整个文档中的大致比重，百分比）
- **key_points**: 关键要点（该章节应覆盖的核心点）

### 4. 特殊要求
- **special_requirements**: 如果文档有特殊格式要求、引用规范、数据要求等，请提取出来

## 输出格式

请按照结构化格式输出，确保：
1. 所有章节的 estimated_percentage 加起来应约等于 100
2. 章节结构应按照文档的自然顺序排列
3. 如果无法确定某个字段，使用合理的默认值
